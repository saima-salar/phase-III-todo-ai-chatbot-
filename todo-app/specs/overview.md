# Todo App - Project Overview

## Project Description
A full-stack todo application with user authentication, task management, filtering, and responsive UI.

## Goals
- Implement secure user authentication with JWT
- Provide full CRUD operations for tasks
- Support task filtering and sorting
- Create responsive UI with dark/light mode

## Non-Goals
- Advanced analytics or reporting
- Team/collaborative features
- File attachments or rich text editing
- Offline synchronization
