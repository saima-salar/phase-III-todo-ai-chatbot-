# Todo App Project Constitution

This document outlines the core principles and guidelines for the Todo App project.

## Core Values

1. **Security First**: All user data must be protected with industry-standard authentication and authorization
2. **User Experience**: Intuitive interface with responsive design
3. **Performance**: Fast loading times and efficient data handling
4. **Maintainability**: Clean, well-documented code following best practices

## Tech Stack Principles

- Backend: FastAPI with SQLModel for type safety and validation
- Frontend: Next.js 14+ with TypeScript and Tailwind CSS
- Database: PostgreSQL with proper ORM handling
- Authentication: JWT-based with proper token management

## Code Quality Standards

- All code must be typed using TypeScript/Python type hints
- Proper error handling for all user inputs and API calls
- Comprehensive documentation for all public APIs
- Consistent code formatting using established tools
