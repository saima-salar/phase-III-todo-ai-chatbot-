# Project Overview

## Todo Full-Stack Web Application

A comprehensive task management application built with Next.js 16+, FastAPI, and Neon Serverless PostgreSQL.

## Features

- Basic task management (Create, Read, Update, Delete)
- Advanced task features (Recurring tasks, reminders, sharing)
- Analytics and dashboard
- Intelligent task suggestions
- User authentication with Better Auth

## Technology Stack

- Frontend: Next.js 16+ (App Router)
- Backend: Python FastAPI
- ORM: SQLModel
- Database: Neon Serverless PostgreSQL
- Authentication: Better Auth
